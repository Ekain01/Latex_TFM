\section{Objetivos}

El objetivo principal es el desarrollo del sistema SCADA para la gestión integral de una red de generación sostenible\par
Para ello se ha de:
\begin{itemize}
\item   Analizar y crear las estructuras de datos necesarias siguiendo la normativa IEC 61400-25-2 para la comunicación en plantas eólicas.
\item   Diseñar la arquitectura del sistema SCADA incluyendo la estructura de comunicación, la base de datos y la interfaz de usuario.
\item   Crear el flujo de navegación de pantallas.
\item   Estudiar el sistema de ciberseguridad IEC 62443 para proteger el sistema SCADA contra amenazas y vulnerabilidades.
\item	Integrar y analizar todas las señales con sus propiedades de comunicación.
\item	Crear sistemas de gestión de alarmas y eventos.
\item	Crear sistemas de análisis de señales analógicas en gráficas de tendencia y datos históricos.
\item Realizar una despliegue en la nube para la visualización de datos en tiempo real.
\item Crear un banco de pruebas. 
\end{itemize}
